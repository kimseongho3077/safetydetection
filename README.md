# 국사
국사는 KT의 다양한 통신 서비스가 원활히 제공될 수 있도록 인프라를 지원하는 곳이며, 건물의 화재 및 문제 발생 시 해당 지역에 막대한 통신 피해를 유발

# 메인 기술
 - YOLOv8
 - Polygon (Shapely 라이브러리) - 다각형을 그려 특정 감시 영역을 설정하고, YOLO 모델로 감지된 객체가 이 영역 안에 있는지를 확인
 - 세션 기반 인증
 - bcrypt 해싱을 사용한 비밀번호 보호
 - Django ORM
